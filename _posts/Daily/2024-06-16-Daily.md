## So What? / So Why?
- 신문이나 잡지를 읽을 때 So What? / Why So?를 연습할 수 있다.
- 지레짐작은 금물
- So What? : 제시한 사실을 전체 집합으로 놓고 그것으로 말할 수 있는 것을 요약하는 작업
- So Why? : 요약된 관찰 결과를 요소 분해하여 검증하는 작업
